Getting Started with Magic Folder

Magic Folder is an intelligent document management system that automatically categorizes and organizes your files using AI. This guide will help you understand how to use Magic Folder effectively.

How Magic Folder Works

Magic Folder monitors a designated "drop" folder on your system. When you place files in this folder, they are automatically:

  • Analyzed using AI to determine the content and context
  • Categorized based on the content
  • Moved to the appropriate category folder
  • Renamed with descriptive names (if enabled)
Key Features
AI-Powered Organization

Files are automatically categorized using advanced AI analysis of their content.

User Feedback System

The system learns from your corrections to improve categorization over time.

Automated Filing

No more manual sorting - files are organized into appropriate folders automatically.

Powerful Search

Quickly find documents based on content, category, or other attributes.

Using the Web Interface

The web interface gives you a convenient way to:

  • Upload files directly through your browser
  • View file statistics and organization
  • Search for specific files
  • Generate reports
  • Configure settings and preferences
Pro Tip: For the best experience, keep both the web interface and the watched folder method available. The web interface is great for individual file uploads and management, while the watched folder is perfect for batch processing.

Uploading Files

There are two ways to add files to Magic Folder:

Web Upload

The web interface allows you to upload files directly through your browser:

  1. Navigate to the Upload page
  2. Drag and drop files into the upload area, or click "Browse Files" to select them
  3. Click "Upload Files" to begin the upload process
  4. Files will be processed and categorized automatically

The web upload supports multiple files at once and provides a progress indicator during upload.

Watched Folder Method

You can also add files by placing them in the watched folder on your system:

  1. Open your file explorer and navigate to the watched folder location
  2. Copy or move files into this folder
  3. Magic Folder will automatically process them

The watched folder location is displayed on the Settings page.

Note: The maximum file size for web uploads is 16MB. For larger files, please use the watched folder method.
Supported File Types

Magic Folder can process and categorize the following file types:

  • Documents: PDF, DOC, DOCX, TXT, RTF, PAGES
  • Images: JPG, JPEG, PNG, GIF, BMP, WEBP, TIFF (with OCR)
  • Spreadsheets: XLS, XLSX, CSV, NUMBERS
  • Presentations: PPT, PPTX, KEY
  • Email: EML, MSG
  • Archives: ZIP, RAR, 7Z (contents are inspected if enabled)

File Management

The Files page provides tools for managing your documents after they've been processed.

Finding Files

There are several ways to find files in Magic Folder:

  • Search: Use the search box at the top of the Files page to search by filename
  • Filter by Category: Select a category from the sidebar to view files from that category only
  • Advanced Filters: Filter by date range, file type, and other attributes
File Actions

For each file, you can:

  • View Details: Click on the file name or the View button to see detailed information
  • Download: Download the file to your computer
  • Recategorize: Move a file to a different category
  • Share: Generate a download link to share with others
  • Delete: Remove the file from Magic Folder
Batch Operations

To perform actions on multiple files at once:

  1. Use the checkboxes to select files
  2. Click the "Batch Actions" dropdown
  3. Select the action you want to perform
File Details

The file details page shows comprehensive information about a file:

  • File metadata (type, size, date added)
  • Category information
  • Content preview (when available)
  • Keywords and entities extracted from the content
  • Similar files based on content analysis
